# 🧩 數獨遊戲

一款功能完整的網頁數獨遊戲，具有多種難度等級、金幣系統、計時功能和排行榜。

## ✨ 功能特色

### 🎯 遊戲核心功能
- **五個難度等級**：初級(⭐) 到 專家(⭐⭐⭐⭐⭐)
- **智能數獨生成**：使用回溯算法生成獨特的數獨謎題
- **答案驗證**：即時檢查數字輸入的正確性
- **視覺反饋**：正確答案綠色顯示，錯誤答案紅色警告

### 📊 統計系統
- **計時器**：精確記錄遊戲時間
- **錯誤計數**：最多允許5次錯誤
- **金幣系統**：正確填入隱藏金幣格子可獲得金幣
- **分數計算**：綜合時間、錯誤次數、金幣和難度的評分系統

### 💰 金幣機制
- 隨機在空格子中放置5-8個隱藏金幣
- 正確填入金幣格子可獲得1-3個金幣
- 金幣會影響最終得分
- 視覺提示：金幣格子右上角顯示💰圖標

### 🏆 排行榜系統
- 分難度等級記錄最佳成績
- 本地存儲保存遊戲記錄
- 顯示完成時間、錯誤次數、金幣數量和總分
- 每個難度等級保留前10名記錄

### 🎮 用戶體驗
- **響應式設計**：適配桌面和移動設備
- **鍵盤支援**：
  - 數字鍵1-9：選擇數字
  - Delete/Backspace/空格：清除格子
  - 方向鍵：移動選擇
  - ESC：暫停/繼續
  - H：顯示提示
- **視覺輔助**：
  - 選中格子高亮
  - 相同數字高亮
  - 相關行列區塊高亮
  - 動畫效果（錯誤震動、金幣收集等）

### 🎨 界面設計
- 現代化漸變背景
- 清晰的數獨網格佈局
- 直觀的數字選擇面板
- 信息豐富的狀態顯示
- 流暢的動畫過渡效果

## 🚀 如何遊玩

1. **選擇難度**：在首頁選擇合適的難度等級
2. **填入數字**：
   - 點擊空格子選擇位置
   - 點擊數字按鈕或按鍵盤數字鍵填入
   - 使用「清除」按鈕或Delete鍵刪除數字
3. **獲得金幣**：留意有💰標記的格子，正確填入可獲得金幣
4. **完成遊戲**：填滿所有格子且不超過5次錯誤即可獲勝
5. **查看排名**：遊戲結束後可查看各難度等級的排行榜

## 🎯 勝利條件
- 填滿整個9x9數獨網格
- 每行、每列、每個3x3方塊都包含1-9的數字
- 錯誤次數不超過5次

## 🚫 失敗條件
- 錯誤次數達到5次

## 💡 遊戲提示
- 使用提示功能會消耗一次錯誤次數
- 暫停遊戲不會停止計時器
- 金幣格子的位置每局遊戲都會隨機生成
- 分數計算公式：時間獎勵 + 金幣獎勵 + 難度獎勵 - 錯誤懲罰

## 🛠️ 技術實現
- **前端**：純HTML5 + CSS3 + JavaScript
- **算法**：回溯法數獨生成和求解
- **存儲**：LocalStorage本地數據持久化
- **響應式**：CSS Grid和Flexbox佈局
- **動畫**：CSS動畫和過渡效果
